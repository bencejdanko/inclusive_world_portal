# inclusive_world_portal

A role-based management portal for the Inclusive World team, designed with Django and [cookiecutter-django](https://github.com/cookiecutter/cookiecutter-django).

## Documentation

Documentation is available at https://inclusiveworld.pasuko.com/docs.

## Contributing

The project is actively looking for contributers of all levels. Please look at our [contribution guide](CONTRIBUTING) for more details on how you can contribute. We are looking mainly for:

- bug testing and issue submission
- submitting pull requests according to our contribution guidelines
