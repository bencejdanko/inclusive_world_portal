# Specification Overview

## What is this system meant to do? 

The Inclusive World Portal is a web and mobile based system used by Inclusive World to manage their Member and Volunteer Data. The Portal will be used for Member and Volunteer Enrollment, Assigning Members and  Volunteers to Programs, Conduct Surveys, track Attendance and collect metrics data.

## Member 

Members are our neurodiverse students. Members will be able to 

- register via portal
- enroll for programs
- pay fees
- confirm their program assignments
- view their enrollment
- view their One Page Description
- view their Attendance history
- view their Survey/Assessment history
- complete surveys

## Volunteers

Volunteers are adults and teenagers who sign up to volunteer with us. Volunteers will be able to:

- register via portal
- enroll yearly for the program
- View Program Information
- view One Page Description
- view Attendance 
- view Survey/Assessment history
- complete surveys.

### Volunteer leads 

Class Lead is an adult volunteer who is a class teacher Class Leads can:

- View Program specific information
- View Member/Volunteer data assigned to Programs they lead
- assign buddies to Members
- track Attendance 
- generate reports
- View Surveys for programs they lead

## Manager 

Managers will:

- view Member and Volunteer Data
- Create Programs
- Assign Members and Volunteers to Programs
- Update One Page Description information
- Create and View Surveys
- generate reports etc

## Person Centered Manager

PCMs will:

- view Member and Volunteer Data, 
- Edit One Page Description tab
- export One Page Description (OPD) as PDF.
- Ability to create, view and generate reports for mid year PCR. 

## Administration

- Manager role plus ability to assign roles
- Add, edit, delete Members and Volunteers.
