# Volunteer

You'll learn from these pages how to: 

- register via portal
- enroll yearly for the program
- View Program Information
- view One Page Description
- view Attendance 
- view Survey/Assessment history
- complete surveys.

## Volunteer Leads

You must be assigned as a volunteer lead by a Manager. As a lead, you have some additional control over programs you lead. You may:

- View and edit program information
- View Member/Volunteer data assigned to Programs they lead
- assign buddies to Members
- track Attendance 
- generate reports
- View Surveys for programs they lead
